# Weather-App

# 🌦️ Weather Web Application

## 📌 Project Overview
📅 **Final Submission Deadline:** 10th April  

This project is a **Weather Web Application** built using **HTML, CSS, and JavaScript**. It demonstrates the use of **API integration, dynamic UI updates, and interactive features**.  

The application allows users to search for any city and view real-time weather information, with a visually responsive UI that changes based on weather conditions.

---

## 🔰 Introduction
The objective of this project is to develop a web application that demonstrates:

- JavaScript fundamentals  
- API integration using `fetch`  
- Dynamic UI rendering  
- Responsive and modern UI design  

This application uses a public weather API to fetch real-time data and display it in a clean interface.

---

## 🌐 API Used
- **OpenWeatherMap API**
- Endpoint: `https://api.openweathermap.org/data/2.5/weather`
- Features:
  - Real-time weather data
  - Temperature, city name, and weather condition

---

## 🚀 Features

### ✅ Core Features
- 🔍 **Search Functionality**
  - Users can search weather by city name  

- 🌡️ **Real-Time Weather Data**
  - Displays temperature and weather conditions  

- 🎨 **Dynamic Background**
  - Background changes based on weather (Clear, Clouds, Rain, etc.)

- 💻 **Responsive Design**
  - Works on mobile, tablet, and desktop  

- ⚡ **Smooth Transitions**
  - UI animations and hover effects  

---

## ⚙️ Technologies Used
